## 1. Комментарии на канвасе
- **Что было:** заглушка-промо в сайдбаре со ссылкой на Plus
- **Что нужно:** полноценная система комментариев
  - Привязка комментария к координатам на канвасе
  - UI: иконка комментария на канвасе + панель/попап с тредом
  - Хранение: localStorage или свой бэкенд
  - Возможно: интеграция с коллаборацией (sync комментариев между участниками)

## 2. Презентации из рисунков
- **Что было:** заглушка-промо в сайдбаре
- **Что нужно:** режим презентации
  - Разбиение канваса на "слайды" (фреймы → слайды)
  - Полноэкранный режим с навигацией (стрелки, клавиши)
  - Анимации переходов между слайдами
  - Экспорт в PDF (опционально)

## 3. AI без лимитов
- **Что было:** rate limit 429 от бэкенда excalidraw.com + кнопка "Upgrade to Plus"
- **Что нужно:** свой AI-бэкенд
  - Поднять прокси-сервер с доступом к OpenAI/Claude API
  - Два эндпоинта:
    - `POST /v1/ai/diagram-to-code/generate` — скриншот → HTML/код
    - `POST /v1/ai/text-to-diagram/chat-streaming` — текст → Mermaid диаграмма (streaming)
  - Обновить `VITE_APP_AI_BACKEND` в `.env` на свой сервер
  - Убрать лимиты или поставить свои

## 4. Облачное хранилище сцен
- **Что было:** Export to Excalidraw+ (Firebase Storage + E2E шифрование)
- **Что нужно:** свой бэкенд для хранения
  - API для сохранения/загрузки сцен
  - Список сцен пользователя
  - Опционально: E2E шифрование, версионирование
